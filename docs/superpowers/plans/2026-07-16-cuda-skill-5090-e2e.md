# CUDA Kernel Optimizer RTX 5090 End-to-End Implementation Plan

> **For AI agent workers:** Required sub-skill: use
> `superpowers:executing-plans` to implement this plan inline. Track every step
> with the checkboxes below.

**Goal:** Prove and improve `cuda-kernel-optimizer` on an RTX 5090 across
Triton, native CUDA, CUTLASS, and one isolated vLLM SM120 workload, then merge
the verified branch into the user's fork `main` only.

**Architecture:** Run opt-in GPU acceptance tests in disposable single-GPU
containers under `/data/tcheng/cuda-skill-e2e`. Preserve raw environment,
timing, Nsight Compute, and SASS artifacts. Every GPU-discovered defect first
becomes a reproducible test; reusable fixes remain in the fork while large
reports remain on the target machine.

**Technical stack:** Python 3.10+, `unittest`, PyTorch CUDA, Triton 3.6 and
3.7.1, CUDA 13.0 and 13.3, Nsight Compute 2025.3.1 and 2026.2.1, CUTLASS 4.6.1,
Docker with one RTX 5090, Git/GitHub CLI.

---

```python
@unittest.skipUnless(os.environ.get("CUDA_SM120_E2E") == "1", "opt-in SM120 test")
class Sm120AcceptanceTests(unittest.TestCase):
    def test_environment_and_fixture_matrix(self):
        env = run_json([PYTHON, CHECK_ENV, "--out", str(ENV_JSON)])
        self.assertEqual(env_file()["primary_sm_arch"], "sm_120")
        for case in CASES:
            result = run_benchmark(case)
            self.assertTrue(result["correctness"]["passed"], case.name)
            self.assertGreater(len(result["kernel"]["samples_ms"]), 4)
            self.assertIn("median_ms", result["kernel"])
            self.assertIn("p95_ms", result["kernel"])
            self.assertIn("cv_pct", result["kernel"])
```

The Triton fixture uses this contract:

```python
def setup(N, seed=None):
    torch.manual_seed(seed or 0)
    return {"inputs": {"x": torch.randn(N, device="cuda"),
                       "out": torch.empty(N, device="cuda"), "N": N},
            "outputs": ["out"]}

def run_kernel(x, out, N):
    vector_kernel[(triton.cdiv(N, 256),)](x, out, N, BLOCK=256)
```

Reference contract:

```python
def reference(x, out, N):
    out.copy_(x * x + 1.0)
```

Native CUDA and CUTLASS fixtures must expose non-const output pointers so the
existing benchmark validates them, and references must mutate the matching
output tensor in place.

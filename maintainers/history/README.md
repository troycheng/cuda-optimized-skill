# Internal development history

This directory contains internal design and implementation history produced
during development. It is retained for maintainers and audit context; Git
history remains authoritative.

It is not the user guide, not the agent execution protocol, and not part of the
public MkDocs navigation. Current user guidance begins at `docs/index.md`; the
canonical Codex protocol remains in `skills/cuda-kernel-optimizer/SKILL.md`.

The directory name describes its audience and purpose. `superpowers` was the
name of a development skill that created some of these files, not a concept in
this project, so it is intentionally absent from the public information
architecture.

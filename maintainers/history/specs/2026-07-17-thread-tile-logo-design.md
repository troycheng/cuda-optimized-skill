# Thread Tile logo 设计

## 目标

为 `cuda-kernel-optimizer` 提供一个简洁、可编辑、适合小尺寸显示的项目标识。图形应能让技术使用者联想到 CUDA 线程块和被优化的核心路径，同时避免使用 NVIDIA、CUDA 或其他项目的官方图形。

## 核心图形

采用已确认的 **Thread Tile** 方案：

- 以 3×3 线程块网格为基础；
- 外围保留 8 个圆角方块，表示并行执行单元；
- 中间方块替换为旋转 45° 的菱形，表示正在被分析和优化的核心路径；
- 不使用芯片外框、文字、循环箭头、渐变、阴影或立体效果；
- 图形在 28 px 下仍应保持九宫格轮廓和中心强调点可辨认。

## 几何规格

基础画布为 `96 × 96`：

- 外围方块：`24 × 24`，圆角半径 `5`；
- 三列和三行的起点分别为 `7`、`36`、`65`；
- 中心菱形顶点为 `(48,34)`、`(62,48)`、`(48,62)`、`(34,48)`；
- 画布四周保留 7 px 安全空间；
- SVG 使用透明背景，不内置圆角底板。

## 配色

采用已确认的 **Carbon + Cyan**：

| 用途 | 外围线程块 | 中心核心 |
|---|---|---|
| 浅色背景 | `#172033` | `#16B8A6` |
| 深色背景 | `#F5F7FA` | `#28D6C2` |

主版本使用浅色背景配色。深色版本只调整填充色，几何形状完全相同。纯单色场景允许把外围和中心统一替换为当前文字颜色。

## 交付文件

- `asset/logo.svg`：浅色背景主版本，透明背景；
- `asset/logo-dark.svg`：深色背景版本，透明背景；
- `asset/logo-128.png`：128×128 透明 PNG；
- `asset/logo-512.png`：512×512 透明 PNG；
- 中英文 README 顶部使用 `<picture>`：浅色模式加载主版本，深色模式加载深色版本，显示宽度为 88 px，并保留现有项目标题和语言切换。

不制作文字标、口号、拟物版本、社交媒体横幅或完整品牌手册。

## 验证

- 两个 SVG 均能作为 XML 解析，`viewBox` 为 `0 0 96 96`；
- 两个 SVG 的几何路径完全一致，只允许颜色不同；
- PNG 尺寸分别为 128×128 和 512×512，并包含透明通道；
- 在白色、`#101826` 深色背景和 28 px 显示尺寸下进行目视检查；
- README 中英文都在浅色模式引用主版本、深色模式引用深色版本，所有本地链接可解析；
- 项目原有自动化测试和 `git diff --check` 继续通过。
